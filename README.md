# Calculator Project

## Overview
This project is a Basic Calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through a simple and interactive user interface.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear screen functionality
- User-friendly interface
- Real-time result calculation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

Calculator/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open `index.html` in a web browser.
4. Use the calculator buttons to perform calculations.

## Working

- Enter numbers using the calculator buttons.
- Select an arithmetic operator.
- Click "=" to display the result.
- Click "C" to clear the display.
